📋 项目概述

AlertBot 是一个现代化的告警管理平台，旨在替代 Prometheus Alertmanager，提供更友好的 Web UI 和强大的告警处理能力。


🚀 现代化架构
• Go + Gin + PostgreSQL 后端
• React 18 + TypeScript + Ant Design 前端
• Docker 容器化部署
⚡ 高性能处理
• 支持 10000+ QPS 告警接收
• 智能去重和聚合
• 实时状态同步
🔔 多样化通知
• 钉钉机器人集成
• 企业微信通知
• 邮件和短信支持
🎯 可视化配置
• 拖拽式规则编辑器
• 直观的告警管理界面
• 实时监控面板

🚀 快速开始

启动服务:

# 启动开发环境
cd web && npm install && npm run dev

API测试:

# 健康检查
curl http://localhost:8080/health

# 运行测试脚本
